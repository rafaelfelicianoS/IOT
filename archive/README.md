# Archive Directory

Esta pasta contém documentação temporária e scripts de teste/debug que foram arquivados durante a limpeza do projeto.

## Estrutura

- **docs/**: Documentação temporária de desenvolvimento (análises, status reports, guias de teste)
- **test_scripts/**: Scripts de teste e debug que foram usados durante o desenvolvimento

## Conteúdo

### docs/
Documentação de desenvolvimento e progresso:
- README.md (versão antiga)
- Análises: ANALISE_COMPLETA.md, ANALISE_FINAL.md
- Status reports: PROGRESS_REPORT.md, PROJECT_STATUS.md, INTEGRATION_STATUS.md
- Implementações: DTLS_IMPLEMENTATION.md, ROUTER_DAEMON_IMPLEMENTATION.md
- Testes: TESTING.md, TESTING_GUIDE.md, MANUAL_TEST_ROUTER_DAEMON.md
- Outros: QUICKSTART, TROUBLESHOOTING, USAGE, CLI, etc.

### test_scripts/
Scripts de teste e debug do BLE:
- debug_*.py: Scripts de debug do Bluetooth/D-Bus
- test_*.py: Scripts de teste individuais
- Ferramentas: diagnose_ble.py, inspect_dbus_gatt.py, clear_bluez_cache.sh

---

**Nota**: Estes arquivos foram preservados para referência histórica mas não são necessários para uso do projeto. Consulte o README.md principal na raiz do projeto para documentação atual.
